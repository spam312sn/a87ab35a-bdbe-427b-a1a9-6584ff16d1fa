API Documentation
=================

## Posts

### Show all posts
URI: `/`

METHOD: `GET`

### Show one post
URI: `/posts/{id}`

METHOD: `GET`

### Create post
URI: `/posts/create/`

METHOD: `POST`

### Delete post
URI: `/posts/delete/{id}`

METHOD: `DELETE`

## Users

### Show all users
URI: `/users/`

METHOD: `GET`

### Show one user
URI: `/users/{id}`

METHOD: `GET`
